# DiscordBirthdayBot
## Description
### A DiscordBot made in python that will send a message for given birthdays

## Usage
  ```sh
  python bot.py
  ```
**Change the token in the python file with your token**
### Commands
  * **Check Latency** - "&ping"
  * **Set Birthday** - "&set_birthday day/month @discordUser"
  * **List birthdays** - "&list_birthdays"
